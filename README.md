# ExerciseFinder

一个使用 SwiftUI 构建的本地健身动作搜索原型。

## 功能

- 搜索动作名称、器械、锻炼部位和目标肌肉
- 按锻炼部位筛选
- 按器械筛选
- 查看动作 GIF、肌群信息和中文步骤
- 所有数据均内置，可离线使用

## 运行

用 Xcode 打开 `ExerciseFinder.xcodeproj`，选择一个 iOS 17 或更高版本的模拟器运行。

## 媒体许可

动作数据来自 `hasaneyldrm/exercises-dataset`。仓库中的图片和 GIF 属于 Gym visual，
当前仅用于本地原型验证。发布或商业使用前必须替换为自有素材，或另行取得 Gym visual 授权。
