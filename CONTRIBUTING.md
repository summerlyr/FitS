# Contributing to FitS

Thanks for helping improve FitS. Bug reports, focused feature proposals,
translation corrections, and search-term improvements are welcome.

## Development setup

1. Clone the repository.
2. Open `ExerciseFinder.xcodeproj` in Xcode.
3. Select the `ExerciseFinder` scheme and an iOS 17 or newer simulator.
4. Build and run the app.

The project has no third-party package dependencies. For a physical device,
select your own development team in Xcode's Signing & Capabilities settings.

## Pull requests

- Keep changes focused and explain the user-facing behavior.
- Build the app before submitting a pull request.
- Include screenshots for visible UI changes when practical.
- Keep Chinese fitness terminology consistent and explain terminology changes.
- Do not submit copyrighted exercise media unless you own it or can document
  permission to redistribute it.
- State the source and license of any new dataset or asset.

By contributing, you agree that your original contributions may be distributed
